HTML Cheat Sheet for Website Development Projects

This HTML Project Cheat Sheet is designed to help beginners become a stronger web developer.

The aim of this document is to provide a guide that's afforable and easy to follow for developing web projects.


Learn HTML for Website Development



HTML is short for "Hyper Text Markup Language"

You can use HTML to create webpages for your website. The document you make can be viewed on the internet by other people.

Using HTML, these cheats will quickly show you how create and markup documents for developing webpages and websites.


TAGS NAME INSTRUCTIONS
<!DOCTYPE html> Doctype This tag is required and must be located at the top of all HTML documents. A closing tag is not required for this element.
<> </> Opening & Closing Angle Brackets Place element inside opening and closing angle brackets. Not all elements need a closing tag.
<html> </html> HTML Element This tag wraps all the content on a webpage. Make sure to add the "lang" attribute for setting the language of the document. Closing tag is required.
<head> </head> Head Element The head element requires an opening and closing angle bracket. This element hold the meta info not viewed on the webpage by the viewer. You should add keywords and a page description to boost search result position. Link a CSS stylesheets to enchance content and deliver a better user experience.
<meta charset="utf-8"> Character Set The character set makes sure your document is set to UTF-8. This meta tag element is nested in the head element. No closing tag required.
<meta name="viewport" content="width=device-width"> Viewport Element Make sure webpage renders at the width of viewport. Stops mobile browsers from rendering webpages wider than the viewport. This meta tag element is nested in the head element. No closing tag required.
<title> </title> Title Element Sets the title of your page. This title will appear in the browser tab of the loaded page, plus it describe the page when you bookmark it or place it in your favorites. Closing tags are required. The title element is nested in the head element.
<body> </body> Body Element Let website vistors discover your text, images, videos, games, playable audio tracks, or other content. Closing tag is required. The body element is placed below the closing head element.

Cheats for Tables

Tables in HTML help you deliver information in a two-dimensional way. That is, by displaying rows and columns of cells containing data.

The HTML elements below will help you cheat your way to creating table.


TAGS NAME DESCRIPTION
<table> </table> Table Element First add <table> </table>. This represents tabular data and is the container for holding information to be displayed in your table. Both opening and closing angle brackets are needed.
<thead> </thead> Table Head Element This step is opitional. Add <thead> </thead>. This give a name to the rows for the head columns of the table. The start tag <thead> is required, but the end tag </thead> may be left out if the <thead> element is immediately followed by a <tbody> or <tfoot> element. You must nest the <thead> </thead> elements inside of the <table> </table> elements.
<tr> </tr> Table Row Element The <tr> </tr> element defines the row of cells in the table. The cells are then established using a mix of <td> </td> (data cell) and <th> </th> (header cell) elements. Nest the <tr> </tr> elements after the opening <table>, <thead>, or <tbody> elements.
<th> </th> Table Header Element The <th> </th> defines the header for a group of table cells. Use the scope attribute.
<td> </td> Table Data Cell Element The <td> </td> element defines a cell which contains data.
<tbody> </tbody> Table Body Element The <tbody> </tbody> element encloses a set of table rows <tr> elements, indicating that they comprise the body of the table (<table>).
<tfoot> </tfoot> Table Footer Element The <tfoot> </tfoot> element defines the rows for the footer columns of the table.
